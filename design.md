# Meetory - 모바일 앱 인터페이스 설계

## 앱 개요
**핵심 컨셉**: 관심사 기반으로 3~5명 자동 매칭 채팅  
**타겟**: 새로운 친구를 만나고 싶지만 부담 없이 소규모 관심사 대화를 원하는 사용자  
**차별점**: 익명 닉네임 + 자동 매칭 + AI 대화 조언

---

## 화면 구조 (Screen List)

### 1. 온보딩 & 인증 플로우
- **로그인/회원가입 화면** (LoginScreen)
  - 이메일 입력 + 비밀번호 입력
  - 회원가입 / 로그인 버튼
  - 간단한 소셜 로그인 옵션 (선택)

- **프로필 설정 화면** (ProfileSetupScreen)
  - 익명 닉네임 입력 (또는 랜덤 생성 버튼)
  - 프로필 사진 선택 (선택사항)
  - 완료 버튼

### 2. 관심사 & 지역 선택 (온보딩 후 처음 1회)
- **관심사 선택 화면** (InterestSelectionScreen)
  - 태그 형태의 관심사 목록 (예: 러닝, 카페, 독서, 요리, 게임, 영화 등)
  - 최소 1개, 최대 5개 선택 가능
  - 다음 버튼

- **지역 선택 화면** (LocationSelectionScreen)
  - 선택식 지역 목록 (강남, 홍대, 건대, 서초, 명동 등)
  - 1개 선택
  - 매칭 시작 버튼

### 3. 메인 탭 네비게이션
- **탭 1: 매칭/채팅 (Home)**
- **탭 2: 발자취 (Footprint)**
- **탭 3: 프로필 (Profile)**

### 4. 매칭 & 채팅 화면
- **매칭 대기 화면** (MatchingScreen)
  - 로딩 상태 표시
  - "관심사: 러닝, 지역: 강남"
  - 취소 버튼

- **채팅방 화면** (ChatRoomScreen)
  - 상단: 채팅방 제목 (예: "러닝 모임 - 강남")
  - 참여자 프로필 미니 카드 (3~5명)
  - 채팅 메시지 리스트
  - 하단: 메시지 입력창
  - **AI 조언 버튼** (💡 아이콘) - 대화 주제 제안
  - 채팅방 나가기 / 재매칭 버튼

- **채팅방 목록 화면** (ChatListScreen)
  - 현재 활성 채팅방 리스트
  - 각 채팅방: 마지막 메시지, 시간, 참여자 수
  - 채팅방 클릭 → ChatRoomScreen으로 이동

### 5. 발자취 화면
- **발자취 메인 화면** (FootprintScreen)
  - 관심사별 인기 장소 TOP 3 표시
  - 예: "러닝 - 한강, 올림픽공원, 강남역"
  - 각 장소 옆에 "체크인" 버튼
  - 내가 체크인한 장소 기록 표시

- **장소 체크인 화면** (CheckInScreen)
  - 선택 가능한 장소 리스트
  - 오늘 방문 장소 선택
  - 체크인 완료 버튼

### 6. 프로필 화면
- **프로필 메인 화면** (ProfileScreen)
  - 닉네임, 프로필 사진
  - 관심사 태그 표시
  - 지역 정보
  - 체크인 통계 (총 방문 장소 수, 최근 활동)
  - 설정 버튼 → SettingsScreen
  - 로그아웃 버튼

- **설정 화면** (SettingsScreen)
  - 관심사 수정
  - 지역 수정
  - 닉네임 수정
  - 차단 목록
  - 신고 기능
  - 앱 정보 / 버전

## 주요 사용자 플로우

### 플로우 1: 신규 사용자 온보딩
1. 로그인/회원가입 → 2. 프로필 설정 → 3. 관심사 선택 → 4. 지역 선택 → 5. 매칭 대기 → 6. 채팅방 입장

### 플로우 2: 채팅방에서 대화
1. 채팅방 입장 → 2. 메시지 입력 → 3. (선택) AI 조언 버튼 클릭 → 4. 대화 주제 제안 표시 → 5. 계속 채팅

### 플로우 3: 채팅방 탈출 & 재매칭
1. 채팅방 → 2. "나가기" 버튼 → 3. 확인 → 4. 매칭 대기 → 5. 새로운 채팅방 입장

### 플로우 4: 발자취 기록
1. 홈 탭 → 2. 발자취 탭 → 3. "체크인" 버튼 → 4. 장소 선택 → 5. 완료

## 핵심 UI 요소

### 1. 채팅 메시지 버블
- 내 메시지: 오른쪽 정렬, 보라색 배경
- 상대 메시지: 왼쪽 정렬, 회색 배경
- 닉네임 표시 (상대 메시지만)
- 타임스탬프

### 2. AI 조언 버튼 & 제안
- 💡 아이콘 버튼 (채팅방 하단)
- 클릭 시 모달 팝업
- 제안 내용: "요즘 러닝 코스 중 가장 인기 있는 곳이 어디예요?"
- 제안을 메시지로 보내기 버튼

### 3. 참여자 프로필 카드 (미니)
- 프로필 사진 (원형)
- 닉네임
- 관심사 1개 표시
- 차단 / 신고 버튼 (롱 프레스 또는 메뉴)

### 4. 발자취 카드
- 관심사 아이콘
- 관심사명
- 인기 장소 TOP 3
- 체크인 버튼

## 레이아웃 원칙 (모바일 포트레이트 9:16)

- **한 손 조작**: 주요 버튼은 화면 하단 중앙 또는 우측에 배치
- **여백**: 모든 요소 간 최소 12px 간격
- **텍스트 크기**:
  - 헤더: 24px (bold)
  - 본문: 16px (regular)
  - 보조: 14px (regular)
  - 라벨: 12px (regular)
- **터치 영역**: 최소 44x44px (Apple HIG)
- **스크롤**: 긴 리스트는 FlatList 사용 (성능)

## 상태 관리 & 데이터 흐름

### 사용자 상태
- `isLoggedIn`: 로그인 여부
- `userProfile`: 닉네임, 프로필 사진, 관심사, 지역
- `currentChatRoom`: 현재 채팅방 정보

### 채팅 상태
- `messages`: 메시지 배열
- `participants`: 채팅방 참여자
- `isLoading`: 메시지 로딩 상태

### 발자취 상태
- `checkIns`: 체크인 기록
- `popularPlaces`: 관심사별 인기 장소
